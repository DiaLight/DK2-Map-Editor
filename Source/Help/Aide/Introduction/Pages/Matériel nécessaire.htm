Pour pouvoir utiliser l'éditeur les composantes suivantes sont nécessaires :

  • Le système d'exploitation Windows® 95/98/Me (ou équivalent).

  • Un processeur de type Pentium® Pro (Famille des Pentium®, Celeron®, AMD®, etc.).

  • DirectX 7.0 ou une version supérieure pour pouvoir jouer les sons

Il n'existe, à ma connaissance, aucune restriction quand au montant de mémoire vive nécessaire. Le programme en lui-même nécessite environ 6Mo (sans compter les structures additionnelles qui sont allouées suivant les actions que vous effectuez). D'une manière générale, un total de 64Mo de mémoire vive devrait suffit.

L'éditeur a été testé avec succès sur les configurations suivantes :

  • Pentium III + nVidia TNT2 Ultra + 256Mo + Win98

  • Celeron + nVidia Geforce + 256Mo + Win98

  • AMD Athlon + nVidia Geforce 2 + 256Mo + WinMe

  • AMD Athlon + nVidia GeForce 4 + 384Mo + WinXP

  • AMD Athlon XP + nVidia GeForce FX + 512Mo + WinXP

Il semblerait que le programme fonctionne avec Windows® 2000 (et probablement avec Windows® NT). J'insiste malgré tout sur le fait que l'éditeur n'a pas été conçu pour être exécuté sur de tels systèmes et que les différences existantes entre l'architecture Win9X (Windows® 95/98/Me) et WinNT (Windows® NT et Windows® 2000) peuvent provoquer des disfonctionnements dans le programme.

 

Ce programme est compatible avec Windows® XP bien qu'il ait été particulièrement conçu pour les versions antérieures de Windows®.
