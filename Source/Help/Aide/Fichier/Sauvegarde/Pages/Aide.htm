Une fois votre travail bien avancé, il peut être utile de sauvegarder votre carte histoire de ne pas tout recommencer depuis le début :) C'est précisément le but des fonctions Sauver et Sauver en...

Lorsque vous débutez la création d'une nouvelle carte, la fonction Sauver est grisée dans le menu Fichier car vous n'avez pas encore spécifié de nom de fichier pour votre carte (le nom de la carte n'est utilisé que lors de la compilation). Elle deviendra accessible après votre première sauvegarde.

Lorsque vous sélectionnez la fonction Sauver en... une boîte de dialogue s'ouvre. Cette boîte de dialogue est similaire à celle que vous ouvrez lorsque vous sélectionnez Ouvrir dans le menu Fichier, la seule différence visuelle étant que le bouton Ouvrir est remplacé par le bouton Enregistrer. De plus, la sélection du format de la carte s'avère nécessaire car c'est ce format qui conditionnera le type de fichier utilisé pour sauvegarder votre carte. Seuls deux formats sont supportés lors de la sauvegarde :

  • Le format de l'éditeur

C'est le format par défaut et il est conseillé de toujours l'utiliser. Les fichiers sauvegardés dans ce format contiennent en effet toutes les informations et les notes que vous avez pu saisir et modifier dans votre carte. Vous êtes ainsi certains de retrouver votre carte telle qu'elle était lorsque vous l'avez sauvegardée.

  • Le format texte

Si vous sélectionnez ce format, la carte sera sauvegardée en fichier texte et pourra être directement lue par l'éditeur de Mark Kinkead (DK2MapperGUI). Toutefois, ne perdez pas de vue que de nombreuses fonctionnalités seront perdues lors de la sauvegarde car le format texte ne supporte pas toutes les options de l'éditeur. Vous perdrez notamment les noms et les couleurs des zones, les noms des bandes de héros, les noms des passages, plus de 90% des paramètres de la carte, les options des créatures, etc.

Lorsque vous sélectionnez ce format, un rapport de compatibilité sera affiché vous indiquant toutes les conversions et les pertes occasionnées par le changement de format.

Vous pouvez cliquer sur certaines lignes du rapport pour voir où est placé l'élément supprimé ou converti sur la carte.
