Pour créer une nouvelle carte, vous devez sélectionner le menu Fichier puis l'entrée intitulée Nouvelle carte... Une fenêtre va alors apparaître.

C’est dans cette fenêtre que vous sélectionnez le nom de la carte (qui sera utilisé lors de la compilation du niveau), ses dimensions ainsi que le type de terrain que la carte contiendra par défaut.

Le nom de la carte est utilisé par le compilateur lors de la sauvegarde des fichiers du jeu. C'est ce nom qui apparaîtra éventuellement lorsque vous choisirez une carte en mode escarmouche ou multi-joueurs.

La version indique à l'éditeur quelle version du jeu est nécessaire pour jouer à cette carte. Il existe 3 versions possibles :

  • "originale" qui est la première version du jeu, dans cette version il ne peut y avoir de créatures élites, de portails des mercenaires et autres objets.

  • "1.61" qui est la version qui a introduit les créatures élites.

  • "1.70" qui est la version qui a introduit le portail des mercenaires, la vierge, et certains objets. Dans cette version, l'objet magique tuer les lutins ennemis  est remplacé par tuer une créature.

La sélection de la version n'est pas fondamentale dans la mesure où l'éditeur compilera toutes les informations même s'il s'agit d'une créature, d'un objet, etc. qui n'est pas utilisable pour la version que vous avez spécifié (un texte vous indiquera juste que certains éléments requièrent une version particulière).

Le nombre de joueurs indique à l'éditeur combien de joueurs se trouvent sur la carte. Là encore, il s'agit d'une simple information qui ne troublera en rien le bon fonctionnement de la carte si elle s'avère inexacte. Certaines fonctions, comme l'ajout automatique des passages, utilisent cette information

Si vous envisagez d’utiliser une carte carrée, cochez la case située à gauche du texte Carte carrée, vous n’aurez alors qu’à modifier l’une des deux dimensions. Mais vous pouvez aussi très bien entrer les mêmes dimensions dans les cases Largeur et Hauteur.

La largeur et la hauteur de votre carte doivent être au moins égales à 7 carrés. Il est de plus recommandé de ne pas créer des cartes dont les dimensions dépassent le 120x120 car le jeu risque de ne plus pouvoir gérer les objets et cela peut occasionner de nombreux ralentissement (quelque soit la vitesse du processeur). Le maximum pour ces deux valeurs est de 300 carrés.

Le nom de la carte, la version, le nombre de joueurs ainsi que les dimensions peuvent être modifiés ultérieurement dans les Propriétés de la carte.
