Ce dernier point du menu Fichier vous permet... de quitter l'éditeur. Si des changements ont été effectués sur la carte, le programme affichera une requête pour s'assurer que vous n'avez pas oublié de sauvegarder vos changements. Si vous répondez oui à cette requête, le programme se terminera (attention! l'éditeur ne demande pas si vous voulez sauvegarder la carte mais si vous voulez bien quitter le programme !).
