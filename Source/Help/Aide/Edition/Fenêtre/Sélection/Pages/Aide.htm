LE RECTANGLE DE SELECTION

 

Le rectangle de sélection est symbolisé sur la carte par un rectangle blanc (voir image ci-dessous). Pour le définir, appuyez sur le bouton droit de la souris à l'endroit où vous désirez le débuter puis déplacez le curseur de la souris pour l'agrandir ou le rétrécir, relâchez ensuite le bouton pour valider le rectangle.

Si vous sélectionnez Tout sélectionner dans le menu Edition (Ctrl+A), le rectangle comprendra toute la carte (à l'exception des limites). Si vous sélectionnez Désélectionner dans ce même menu (Ctrl+N), le rectangle sera supprimé de la carte.

Le rectangle de sélection est utilisé dans de nombreuses fonctions d'édition.
