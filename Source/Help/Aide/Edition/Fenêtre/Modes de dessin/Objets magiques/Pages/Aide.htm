• Les objets magiques

Pour placer un objet magique sur la carte, commencez par sélectionner le mode de dessin Placer un objet magique. Par défaut, l'éditeur placera une augmentation de niveau appartenant au joueur neutre. Pour changer l'objet magique, cliquez une nouvelle fois sur le bouton de sélection du mode de dessin ou appuyez sur le bouton droit de la souris lorsque le curseur se trouve au-dessus de ce bouton.

Parmi les objets magiques vous trouverez :

  • Augmenter le niveau.

    Ajoute un niveau d'aptitude à toutes les créatures détenues par le joueur.

  • Révéler la carte.

    Cette objet supprime le brouillard de guerre. La carte devient alors entièrement visible.

  • Fortifier les murs.

    Tous les murs du joueur sont fortifiés.

  • Supprimer les fortifications ennemies.

    Tous les murs du joueur ennemi sont détruits.

  • Augmenter l'or.

    Ajoute de l'or au joueur (par défaut, 10.000 unités).

  • Augmenter le mana.

    Ajoute du mana au joueur (50.000 unités par défaut).

  • Assommer les lutins ennemis.

    Tous les lutins ennemis sont stoppés pendant une courte période.

  • Main d'oeuvre magique.

    Ajoute des lutins au joueur (10 par défaut).

  • Rendre les créatures heureuses.

    Rend toutes les créatures du joueur heureuses.

  • Rendre les créatures tristes.

    Rend toutes les créatures du joueur ennemi tristes.

  • Tuer les lutins ennemis.

    Tue tous les lutins appartenant à l'ennemi. Attention! Dans la version 1.7, cet objet magique est remplacé par Tuer une créature de l'ennemi et il permet de tuer directement la créature désignée par le joueur.

  • Redonner la santé aux créatures.

    Guéris toutes les créatures appartenant au joueur.

  • Accéder à un niveau secret.

    Permet au joueur d'accéder à un niveau secret dans la campagne. Cet objet magique n'a aucun intérêt dans les niveaux que vous créez.

Comme pour les objets, les objets magiques appartiennent au joueur neutre par défaut. Modifier le détenteur d'un objet magique n'est normalement pas nécessaire.

Sur la carte, les objet magiques apparaissent sous la forme de losange bleu ou de la couleur du joueur si vous avez sélectionné Utiliser les couleurs des joueurs pour les objets magiques dans le menu afficher.
