• Les objets

Les objets se placent comme les créatures, les portes et les pièges. Sélectionnez l'icône Placer un objet dans la barre d'outils. Par défaut, l'éditeur placera un tas d'or appartenant au joueur neutre et contenant 0 unités. Pour modifier l'objet, cliquez à nouveau sur l'icône ou appuyez sur le bouton droit de la souris.

Vous pouvez placer un nombre important d'objets, certains ont une finalité purement esthétique, d'autres peuvent s'avérer utiles pour le joueur. Les objets les plus communément utilisés sont :

  • L'or, le sachet d'or et le coffre d'or. Ils permettent notamment de remplir les salles du trésor appartenant au joueur neutre et aux héros. Dans les salles du trésor, le coffre d'or doit impérativement contenir 3.000 unités (c'est le maximum par défaut qu'un carré de salle du trésor peut contenir) sinon il ne s'affichera pas. Vous n'êtes cependant pas obligé de placer ces objets dans une salle du trésor, vous pouvez très bien les placer sur n'importe quel type de terrain. Pour modifier la quantité d'or contenue dans ces objets, il suffit de remplacer la valeur située à droite du texte intitulé "Quantité d'or".

  • Le livre. Le livre permet de subordonner la découverte d'un sort à la découverte d'un livre donné. Ainsi, lorsqu'un joueur prendra le contrôle du carré sur lequel se trouve le livre, un de ses lutins ira porter le livre dans une bibliothèque occulte et le sort contenu dans ce dernier se trouvera alors accessible. Vous pouvez sélectionner le sort en cliquant sur le bouton situé à droite du texte intitulé "Contenant le sort"; sélectionnez ensuite le sort désiré et appuyez à nouveau sur le bouton gauche de la souris. Le livre que vous placerez alors sur la carte contiendra le sort sélectionné.

  • Le cristal. Le cristal est généralement l'objet à récupérer pour terminer le niveau. Pour le faire apparaître sur la carte, il est non seulement nécessaire de placer l'objet correspond mais il faut en plus utiliser un évènement. Cela permet de cacher le cristal jusqu'à ce qu'un objectif déterminé soit réalisé. La carte ne peut contenir qu'un seul cristal.

  • Les coffres de pièges et de portes. Les coffres permettent au joueur de pouvoir placer directement un piège ou une porte sans avoir à attendre qu'elle soit construite dans l'atelier. Le joueur a toutefois besoin d'un atelier pour pouvoir placer ces éléments.

  • Les armes. Il s'agit d'objets ayant un but purement esthétique. Ils permettent de donner un peu plus de consistance aux châteaux de héros.

  • Les lits. Comme pour les armes, ces objets n'ont qu'un but esthétique. Ils n'apparaissent que s'ils sont placé dans un antre.

Il existe bien d'autres objets mais notez que certains ont une durée de vie limitée (comme les poulets) et que d'autres ne se placent pas nécessairement comme vous le voudriez (il n'est pas possible de définir l'orientation d'un objet donc ils sont tous orientés de la même manière).

Par défaut, les objets appartiennent au joueur neutre. En règle générale, il n'est pas nécessaire de modifier le détenteur d'un objet.

Sur la carte, les objets apparaissent sous la forme de carrés rouges ou sous la forme de carré de la couleur du joueur si vous avez sélectionné l'option Utiliser les couleurs des joueurs pour les objets dans le menu afficher.
