LE CURSEUR

 

Lorsque vous déplacez le pointeur de la souris sur la carte, vous vous apercevrez qu'un carré bleu le suit continuellement. Ce carré indique l'endroit où vous allez dessiner. Il peut parfois prendre la forme d'un rectangle plus important selon l'élément que vous désirez dessiner. Par exemple, les coeurs de donjon utilisent OBLIGATOIREMENT 5 carrés par 5 donc le carré aura 5 carrés de largeur et 5 carrés de hauteur.

Le carré peut également prendre différentes couleurs suivant le terrain au-dessus duquel il se trouve et suivant ce que vous vous apprêtez à faire. Dans tous les cas, si le carré devient rouge, cela signifie que vous ne pouvez pas dessiner à l'endroit sélectionné (figure 1).

Figure 1

Si vous avez sélectionné un mode de dessin, le carré peut devenir bleu clair lorsqu'il se trouve sur un portail (figure 2).

Figure 2

Le carré peut également devenir bleu clair si vous tentez de placer une créature sur un terrain qui lui est fatal (figure 3).

Figure 3

Enfin, le carré devient vert si vous tentez de placer une créature, ou un objet quelconque sur un terrain ennemi (figure 4).

Figure 4

Lorsqu'un élément a une taille minimale ou maximale, la sélection du mode de dessin n'est pas totalement prise en compte. Toutes les formes vides se comportent comme si vous aviez sélectionné une forme pleine (le rectangle défini est toujours remplis, quelque soit le mode sélectionné); de plus, il n'est pas possible de faire des ellipses, un rectangle sera dessiné à la place; enfin, le remplissage se comporte comme si vous aviez sélectionné le stylo.

Lorsqu'un élément a une taille fixe, la sélection du mode de dessin est totalement ignorée. L'éditeur fera comme si vous aviez sélectionné le stylo.

Voir les modes de dessins pour de plus amples informations.
